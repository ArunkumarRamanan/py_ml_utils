Python ML Utilities - py_ml_utils
=================================

Tools
--------

- FeatureTransformer
- MissingValueInferer
- DatasetTransformer
- GreedyOptimizer
- FeatureSelector

Software Requirements
---------------------

- Python 2.7 or >= 3.4
- Numpy >= 1.13.1
- Scipy >= 0.19.1
- Scikit-learn >= 0.19
- Pandas

Installation
------------

From `GitHub <https://github.com/goldentom42/py_ml_utils>`__:

::

   git clone https://github.com/goldentom42/py_ml_utils.git
   python setup.py install
